# IJCO Key Messages

1. A fully reproducible public-data workflow identified stable cross-layer ovarian cancer hubs (LF7/LF8/LF6).
2. Network and perturbation evidence was stronger and more consistent than advanced predictive-model discrimination.
3. Fairness-controlled benchmarking (protein-matched analysis) prevented inflated claims from sample-overlap effects.
4. The outputs are directly usable for translational follow-up: hub prioritization, pathway testing, and external cohort validation.
