# BJC Upload Order

1. Main manuscript: `BJC_Main_Manuscript.docx` or `BJC_Manuscript_Submission.docx`
2. Title page with author, affiliation, corresponding email, and ORCID
3. Cover letter: `BJC_Cover_Letter_Final.docx`
4. Supplementary appendix: `BJC_Supplementary_Appendix.docx`
5. Supplementary support notes: `hub_interpretation.md`, `hub_pathway_enrichment.md`
6. Figure files if journal submission system requests separate artwork
7. Any journal declarations entered in the submission portal
